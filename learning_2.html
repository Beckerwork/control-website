Daten sammeln und speichern

Die Daten, die von großen Unternehmen täglich gesammelt werden, werden in großen Räumen auf sogenannten Servern gespeichert. In einem Serverraum stehen viele Schränke, die voll mit blinkenden Computern. Oft sind diese speziellen Räume sehr gut gekühlt, damit die Technik stabil arbeiten kann.

Es werden zum Beispiel Bewegungsprofile, Schlafrhythmen und soziale Verbindungen gespeichert. Damit können Profile von einzelnen Menschen zusammengebaut werden. Diese Daten könnten z.B. über eine Smartwatch oder Soziale Medien gesammelt werden.

Die Daten werden meistens sicher und verschlüsselt aufbewahrt, damit niemand Unbefugtes Zugriff darauf bekommt. Es passiert aber leider immer wieder, dass Unbefugte Zugriff auf diese Daten bekommen und sie im schlimmsten Fall veröffentlichen.

Das Vorhersageproblem

Computersysteme können über Sensoren mittlerweile viele Dinge erkennen. Was jeder kennt, sind beispielsweise Bewegungsmelder, die Bewegungen erkennen können. Es gibt auch Wärmesensoren, die Temperaturen in der Umgebung erkennen können. Außerdem gibt es Kamerasysteme, welche die Größe von Dingen in ihrem Sichtfeld berechnen können. Zahlreiche Systeme in unserem Alltag nutzen solche Sensoren. Man kann seinen Puls messen, die Geschwindigkeit im Auto oder seinen Standort in Navigations Apps anzeigen lassen. Diese Sensoren messen etwas in einem Moment und geben dann eine Information zurück.

Aber es gibt auch Technologien, die es Systemen ermöglichen, Dinge vorherzusagen, die nicht gerade im Moment messbar sind. Dafür werden gespeicherte Daten genutzt, mit denen Berechnungen durchgeführt werden.

Stell dir vor, ein System soll vorhersagen, wie schnell ein Kind rennen kann. Das ist möglich, basierend nur auf der Körpergröße eines Kindes und einem Datensatz, der über ganz viele Kinder gespeichert worden ist. Es ist zwar keine genaue und mit Sicherheit passende Vorhersage möglich, aber es kann ein Wert geschätzt werden.

Angenommen, es wurden Daten von vielen Kindern gesammelt. Ihre Körpergröße (in cm) und ihre maximale Laufgeschwindigkeit (in km/h). Diese Daten können in einem Diagramm dargestellt werden.

Auf der horizontalen Achse nach rechts (x-Achse) steht die Körpergröße, auf der vertikalen Achse nach oben (y-Achse) die Geschwindigkeit.

Die Vorhersagetechnik

Jeder Punkt im Diagramm steht für ein Kind. Dabei wurden zwei Dinge gemessen. Die Körpergröße und die maximale Laufgeschwindigkeit.

Um mit diesen Datenpunkten Vorhersagen zu machen, wird eine Gerade durch diese Punkte gelegt. Die Gerade muss so gelegt werden, dass sie möglichst nah an allen Datenpunkten liegt. Mit dieser Geraden kann dann für jede beliebige Körpergröße eine Geschwindigkeit vorhergesagt werden. Auch wenn eine bestimmte Größe nicht im Datensatz vorkommt.

Beispiel: Angenommen, es soll die Geschwindigkeit für ein Kind mit einer Körpergröße von 145 cm vorhergesagt werden. Bei einer Größe von 145 cm gibt es keinen Datenpunkt. Die Gerade läuft dann trotzdem durch den Bereich. Dann kann damit eine Vorhersage getroffen werden. So eine Gerade nennt man eine Regressionsgerade. Was Regression bedeutet, wird später erklärt.

Aufgabe 2.1 – Regressionsgerade erstellen

Jetzt bist du dran. Versuche, eine Gerade zu finden, die möglichst gut zu den Datenpunkten passt.

Verändere die Steigung und den y-Achsenabschnitt deiner Geraden so, dass sie möglichst nah an allen Datenpunkten liegt. Je kleiner der Fehlerwert, desto besser.

Die durch Machine Learning berechnete Regressionsgerade kannst du zum Vergleich durch einen Klick auf „Zeige Regressionsgerade" anzeigen lassen. Aber nur, wenn deine Gerade schon genau genug ist!

💡 Was ist der Fehlerwert?

Der Fehlerwert zeigt, wie weit die Datenpunkte im Durchschnitt von der Geraden entfernt sind. Je kleiner der Wert, desto besser passt die Gerade zu den Daten und desto präziser sind die Vorhersagen.

Finde heraus: Wie schnell ist laut der Regressionsgerade ein Kind mit einer Körpergröße von 145 cm?
